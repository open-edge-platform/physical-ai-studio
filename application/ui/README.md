# Geti Action UI

React web application for the Geti Action studio.

## Setup

```bash
npm install
```

## Development

```bash
# Start dev server
npm run start

# Build for production
npm run build

# Preview production build
npm run preview
```

## Testing

```bash
npm run format
npm run lint
npm run type-check
npm run test:unit
npm run test:component
```
